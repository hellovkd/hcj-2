Web programming

HTML, CSS3로 개인페이지 만들기

HTML

HTML은 웹 문서를 구조화하는 마크업 언어이며, CSS는 문서의 스타일을 디자인하는 스타일 시트 언어입니다. 이 두 가지 기술을 사용하여 웹 페이지의 모양과 느낌을 제어할 수 있습니다.

CSS

HTML은 웹 문서를 구조화하는 마크업 언어이며, CSS는 문서의 스타일을 디자인하는 스타일 시트 언어입니다. 이 두 가지 기술을 사용하여 웹 페이지의 모양과 느낌을 제어할 수 있습니다.

부트스트랩 (Bootstrap)

부트스트랩은 반응형 웹 디자인 및 개발을 위한 무료 오픈 소스 프론트엔드 프레임워크입니다. 부트스트랩은 HTML, CSS 및 JavaScript를 사용하여 웹 개발을 빠르고 쉽게 할 수 있도록 도와줍니다. 또한 다양한 디자인 및 레이아웃 옵션을 제공하여 일관된 외관과 느낌을 가진 웹 페이지를 구축할 수 있습니다.

테일윈드 (Tailwind CSS)

테일윈드는 사용자 지정 스타일 시트를 작성하는 대신 미리 정의된 클래스를 사용하여 웹 페이지를 디자인하는 CSS 프레임워크입니다. 테일윈드를 사용하면 HTML에 직접 클래스를 추가하여 스타일을 지정할 수 있으며, 빠르고 일관된 디자인을 유지할 수 있습니다. 테일윈드는 커스터마이징이 쉽고 확장성이 높은 CSS 프레임워크로, 다양한 프로젝트에 유용하게 사용될 수 있습니다.
